# License Plate Scanner

A deep learning project for automatic license plate recognition (ALPR). Built for CSC671.

**Team:** David Lei, Luis Gabriel Ortiz-Anguiano, Ryan Kurian George

---

## Overview

This project trains a model to detect and read license plates from images, extracting the plate number and identifying the US state.

## Tech Stack

- **Deep Learning:** PyTorch
- **Computer Vision:** OpenCV

## Dataset

10,125 high-quality license plate images used for training and evaluation.
